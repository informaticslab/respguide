Introduction
============

    The mToxGuide was the first mobile prototype done in the Informatics R&D Lab. It takes the information from the CDC Tox Guides and put it in a mobile application. 

Installation and Building
=========================

    The mToxGuide source code was built using Xcode 4.3.2 running on Apple Mac OS X Lion 10.7. The application was targeted for use on the iPhone or iPod Touch iOS devices.
